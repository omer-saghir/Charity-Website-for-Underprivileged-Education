# Charity Website for Underprivileged Education

This is a responsive, multi-page web platform dedicated to supporting the schooling of underprivileged children. Developed using **HTML5** and **CSS3**, the project provides a transparent and engaging interface to connect donors, volunteers, and corporations with educational initiatives.

# Features

### User Navigation & Experience
* **Interconnected Architecture:** Seamless navigation between Home, About Us, Donate, and Contact Us pages.
* **Responsive Design:** Fully mobile-friendly layout optimized for various screen sizes using CSS Media Queries.
* **Interactive UI:** Custom hover effects on buttons and links to enhance user engagement.
* **Clean Layout:** Minimalist design focusing on readability and a positive, welcoming atmosphere.

### Donation & Volunteer Management
* **Sponsorship Platform:** Dedicated sections for users to sponsor a child's education or contribute to specific campaigns.
* **Volunteer Integration:** A system for individuals to sign up as mentors or teachers to contribute their time and skills.
* **Impact Transparency:** Showcases success stories and charitable activities to build trust with individual and corporate donors.
* **Call to Action:** Prominent and stylized buttons designed to drive contributions and sign-ups.

### Design & Implementation
* **Semantic HTML:** Structured using modern HTML5 tags (header, nav, section, footer) for better accessibility.
* **Custom Styling:** Use of professional typography (Arial/Georgia) and a cohesive color scheme.
* **Accessibility Support:** Implementation of ARIA labels and keyboard navigation to ensure inclusivity for all users.
* **Visual Storytelling:** Integrated high-impact graphics and images to connect emotionally with the visitors.

# Technical Stack

* **Frontend:** HTML5, CSS3
* **Tools:** Visual Studio Code / Notepad++
* **Concepts:** Flexbox/Grid, Responsive Web Design, Media Queries, UI/UX Principles

# Prerequisites

* **Modern Web Browser:** (Google Chrome, Mozilla Firefox, or Microsoft Edge)
* **Text Editor:** For code modifications (VS Code)
